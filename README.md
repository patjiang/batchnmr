# batch-nmr
### In order to Use this code, please ensure your data is structured as follows:

```bash
file_dir/
├── sample name/condition (1)
├    ├── key (1)
├    └── key (2)
├── sample name/condition (2)
├    ├── key (1)
├    └── key (2)
├── sample name/condition (3)
├    ├── key (1)
├    └── key (2)
└──sample name/condition (4)
     ├── key (1)
     └── key (2)
```
